## 部署

请先将 bin/deploy_to_github.sh 中的 `git@github.com:FrankFang/react-mangosteen-1-preview.git` 替换为你的仓库地址，然后在 bash 中执行：
```bash
bin/deploy_to_github.sh
#或
sh bin/deploy_to_github.sh
```
